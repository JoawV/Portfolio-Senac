MEUS PROJETOS

Biblioteca

Desenvolvi um projeto no NetBeans usando Java para simular a interface de um aplicativo de biblioteca. Criei uma representação visual com botões, campos e menus, permitindo aos usuários interagir com a biblioteca. Embora seja uma simulação e não esteja conectado a um banco de dados real, o projeto me deu prática na criação de interfaces gráficas.
