MEUS PROJETOS

Biblioteca

Desenvolvi um projeto no NetBeans usando Java para simular a interface de um aplicativo de biblioteca. Criei uma representação visual com botões, campos e menus, permitindo aos usuários interagir com a biblioteca. Embora seja uma simulação e não esteja conectado a um banco de dados real, o projeto me deu prática na criação de interfaces gráficas.

RPG

Desenvolvi um projeto usando TypeScript para simular um jogo de RPG, criando mecânicas de personagens, combates e interações típicas desse gênero. Embora uma simulação simplificada, o projeto permitiu a aplicação de conceitos de programação de jogos e lógica, além de me familiarizar com o uso do TypeScript para desenvolvimento de aplicações interativas.

Zumbi

Criei um projeto simulando um jogo de sobrevivência em um cenário apocalíptico de zumbis. O ambiente virtual envolveu desafios como enfrentar zumbis, coletar recursos e tomar decisões de sobrevivência. Essa experiência prática me permitiu explorar programação e design de jogos em um contexto temático de apocalipse zumbi.
