Resumo

Neste projeto, criei uma simulação de interface gráfica para um programa de biblioteca usando a linguagem Java e o ambiente NetBeans. Através dessa simulação interativa, desenvolvi uma representação visual que permitia aos usuários explorar as funcionalidades de um aplicativo de biblioteca. Embora não estivesse conectado a um banco de dados real, o projeto me deu prática na criação de interfaces interativas e na aplicação de lógica de programação em ambientes visuais. O uso do NetBeans facilitou o desenvolvimento e me proporcionou uma experiência valiosa em design de interface e programação Java.
